# 利用 Python 进行 NBA 比赛数据分析  

## 实验知识点

- nba 球队的`Elo score`计算
- 特征向量
- 逻辑回归  

环境为`python 2.7.x`  

## 实验流程

本次课程我们将按照下面的流程实现** NBA** 比赛数据分析的任务：  

1. 获取比赛统计数据  
2. 比赛数据分析，得到代表每场比赛每支队伍状态的特征表达  
3. 利用**机器学习**方法学习每场比赛与胜利队伍的关系，并对 2016-2017 的比赛进行预测
